# 🚀 快速开始指南

## 5分钟上手 - 使用在线API（推荐）

### 步骤1: 安装项目（1分钟）

```bash
git clone https://github.com/0qinghao/AI_chat_guardian.git
cd AI_chat_guardian
pip install -r requirements.txt
```

### 步骤2: 获取免费API密钥（2分钟）

1. 访问 **智谱AI**：https://open.bigmodel.cn
2. 注册/登录（支持微信、手机号）
3. 进入「API密钥」页面
4. 点击「创建新的API密钥」
5. 复制密钥（格式：`a1b2c3d4e5f6...`）

**免费额度**：100万tokens/月（约50万汉字），足够日常使用！

### 步骤3: 配置API密钥（30秒）

```powershell
# Windows PowerShell（推荐）
$env:ZHIPU_API_KEY="粘贴你的密钥"

# 验证是否设置成功
$env:ZHIPU_API_KEY
```

### 步骤4: 修改配置文件（30秒）

编辑 `config/default_config.yaml`，找到 `llm_detector` 部分：

```yaml
llm_detector:
  type: api        # 改为 'api'
  enable: true     # 确保为 true
  threshold: 0.7
```

### 步骤5: 测试运行（1分钟）

```bash
# 快速测试
python test_api.py

# 启动GUI
python gui.py
```

## 💡 常见问题

### Q: 提示"API密钥未设置"？
```powershell
# 重新设置环境变量
$env:ZHIPU_API_KEY="你的密钥"

# 检查是否设置成功
$env:ZHIPU_API_KEY
```

### Q: API调用失败？
1. 检查网络连接
2. 确认API密钥是否正确
3. 查看控制台错误信息
4. 尝试重新生成密钥

### Q: 免费额度用完了？
1. 切换到其他提供商（硅基流动）
2. 等待下月额度重置
3. 切换回本地模式
4. 降低检测频率

### Q: 如何切换到其他API提供商？

**硅基流动**（多模型选择）：
```yaml
llm_detector:
  api:
    provider: siliconflow
```
```powershell
$env:SILICONFLOW_API_KEY="你的密钥"
```
注册：https://siliconflow.cn

## 🎯 推荐配置（复制即用）

```yaml

detection:
  enable_regex: true
  enable_keyword: true
  confidence_threshold: 0.7

llm_detector:
  type: api
  enable: true
  threshold: 0.7
  api:
    provider: zhipu
    api_key: ""  # 从环境变量读取

```

```powershell

$env:ZHIPU_API_KEY="your_api_key_here"
```
